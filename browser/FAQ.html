FAQ for CisGenome Browser: A flexible tool for genomic data visualization

Hui Jiang

What is the license for CisGenome Browser?

There is no license require for non-commercial use of CisGenome Browser, which means anyone can use the source codes, documents or the excutable file of CisGenome Browser free of charge. For commercial use, please contact the author.

What platform does CisGenome Browser run on?

CisGenome Browser runs on Windows, Linux and Mac systems.

What web browsers does CisGenome Browser work with?

CisGenome Browser works with most of the web browsers, IE, Firefox, Safari, etc.

What programming language is it written in?

It is written in C++.

How to hide a session from the starting page?

All sessions with the names starting with "_" will be hidden automatically.

Why are there some buttons which are disabled?

Not all functions of the browser can be used remotely.

How to put a multiple line title for a track?

You can put html tags such as <BR> or <P> inside a title to achieve that.

How do I cite CisGenome Browser?

Please use the following reference to cite CisGenome Browser:

Jiang, H., Wang, F., Dyer, N.P., Wong, W.H. (2010)
CisGenome Browser: A Flexible Tool For Genomic Data Visualization
Bioinformatics, in press.
